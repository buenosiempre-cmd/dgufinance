## 왜 지금 변화해야 하는가

재무·회계·세무는 **전문지식 의존도가 가장 높은 업무**입니다. 전문 부서일수록 일하는 방식을 먼저 바꿔야 AI 시대에 대응할 수 있습니다.

**현재 문제점:**
- 자료가 흩어져 있다 → 업무 매뉴얼이 그룹웨어·nDRIMS·개인 PC에 산재, 검색 불가
- 사람이 바뀌면 지식이 사라진다 → 담당자 이동·휴직 시 업무 노하우 유실
- 같은 질문에 반복 답변 → 원천징수율, 연말정산 절차 등 매년 같은 문의에 전화·이메일 대응

## 1. 소통 방식 혁신 — Slack ✅

**Slack이란?** 비즈니스용 메신저 앱. 주제별 채널로 대화를 분리하고, 모든 대화·파일이 영구 보존·검색됩니다.

| Before (카카오톡) | After (Slack)       |
| ------------- | ------------------- |
| 대화 검색·축적 불가   | 모든 대화·파일 영구 검색      |
| 파일 30일 만료     | 업무별 채널 분리 운영        |
| 업무·잡담 혼재      | 스레드로 주제별 맥락 유지      |
| 퇴사 시 히스토리 소멸  | AI 도구(Claude) 연동 가능 |

**실제 활용 사례:**
- 채널 분리 운영: `#00-공지사항` `#02-상세질문과토론` `#04-인건비` `#05-서류검토` 등
- 연말정산센터 Slack 운영: 학생 근로자 7명과 채널 기반 민원 접수·처리
- Google Calendar 연동 → 주간 일정 자동 공유, Claude 연결로 채널에서 직접 업무 질의

## 2. 지식 관리 혁신 — Obsidian + 마크다운 🔴

**Obsidian이란?** 마크다운(.md) 파일 기반 **개인 지식 관리 도구**. 문서 저장소를 「Vault(볼트)」라 부르며, 담당자 개인 작업 공간으로 사용합니다. 완성된 매뉴얼은 Notion에 게시하여 팀 전체가 공유합니다.

### 왜 마크다운인가?

- 마크다운(.md) = 제목·문단·목록 등을 단순 기호로 표현 → **사람과 AI가 함께 읽기 쉬운** 문서 형식
- HWP/PPT/PDF는 글꼴·자간 등 편집 요소 때문에 AI가 문서 구조를 정확히 인식 불가
- **국제적으로 AI 학습에 주로 사용**되는 표준 문서 형식

> **국가AI전략위원회**(2026.3.5 발표) — 분과별 회의·토론 결과 문서를 마크다운으로 작성·관리·공개 전환. 기존 HWP 문서가 시각적 표현 중심이라 AI 활용에 부적합하다는 인식에서 추진.

> 💡 **이 보고서 자체가 마크다운으로 작성되었습니다.** `#` 제목, `-` 목록, `|` 표, `>` 인용 등 단순 기호만으로 구조화된 문서를 만들 수 있습니다.

### Vault 폴더 구조 - (예시) 재무팀 업무분장 기반 5대 영역

```
📁 Vault (= 담당자 개인 문서 저장소)
├── 📁 결산·자금 — 연결산·월결산, 자금운용, 기금운용심의
├── 📁 등록(수입) — 등록금 수납·환불, 가상계좌, 선급장학금
├── 📁 회계(지출) — 지출 검토·승인, 법정지출증빙, 법인카드
├── 📁 세무 — 인건비, 연말정산, 원천세, 부가세, 세무자문
├── 📁 프로젝트 — 퇴직연금·디지털혁신 등 진행 중 과제
└── 📁 템플릿 — 반복 사용 문서 틀
```

**실제 활용 사례:**
- 설명회 진행 스크립트, 업무 매뉴얼 등을 마크다운으로 구조화 (단계별 절차 + 실무 이슈 반영)
- Claude Desktop에 Vault를 연결 → AI가 매뉴얼 전체를 읽고 업무 질문에 답변 초안 생성

## 3. 정보 공유 혁신 — Notion 🟠

**Notion이란?** 팀 협업용 올인원 워크스페이스 (문서 + 데이터베이스 + 프로젝트 관리). 모바일·PC 어디서든 접근 가능합니다.

### 도구 간 흐름

```
Obsidian (개인 초안) → Notion (팀 공유) → Slack (링크·피드백) → Claude (AI 답변)
```

**실제 활용 사례:**
- **재무팀 지식 라이브러리** : 내부 매뉴얼 + 교직원 FAQ 이중 구조의 통합 지식 허브
- **재무팀 안내 포털** : 연말정산 안내 포털, FAQ 및 결과 안내, 결산 안내, 등록금 수납안내 등 안내 통합. 연말정산 FAQ를 구글폼 상단에 삽입하여 교직원 배포
- **매뉴얼 데이터베이스 (구축 예정)** : 업무영역·담당자·상태·관련 시스템별 검색·필터링

## 4. AI 활용 혁신 — Claude 🟣

**Claude란?** Anthropic사 개발 AI 대화형 어시스턴트. 긴 문서 이해·분석 및 바이브 코딩 등에 강점이 있어 업무 매뉴얼 기반 질의응답에 적합합니다.

| Before (수동 탐색)       | After (AI 활용)          |
| -------------------- | ---------------------- |
| 같은 질문 매달 수십 건 전화·이메일 | AI가 매뉴얼 기반 답변 초안 자동 생성 |
| 매뉴얼 위치를 아는 사람만 검색 가능 | 구조화된 질문으로 답변 정확도 향상    |
| "그건 OOO씨가 알아요" → 속인화 | 팀 지식이 AI 컨텍스트로 전환      |
| 신규 직원 적응 수개월         | 24시간 검색 가능, 경험 갭 해소    |

**실제 활용 사례:**
- **Claude + Obsidian Vault 연동** : "시간강사 원천징수 방법"을 묻으면 관련 매뉴얼을 찾아 답변 초안 생성
- **Slack + Claude 연동** : 채널에서 Claude를 호출하여 Calendar·Drive·Gmail까지 원스톱 처리
- **세법 검토 자동화** : 세무자문 DB를 바탕으로 세법 간련 사례 검토 자동 수행

> ⚠️ **핵심 원칙: AI 사용자는 재무팀**
> - 교직원이 AI를 직접 사용하는 것이 아님
> - 재무팀이 AI로 답변 초안 생성 → 전문가가 검토·보완 → 최종 답변 제공
> - **전문성 + AI** 결합으로 응답 속도와 품질을 동시에 확보
