## 1. Slack — 업무용 카카오톡 ✅ 전환 완료

카카오톡으로 업무를 하면 대화가 묻히고, 파일이 만료되고, 사람이 나가면 기록이 사라집니다. Slack은 이 문제를 해결합니다.

**카카오톡과 뭐가 다른가요?**
- **채널** = 주제별 대화방. `#결산` `#출납` `#인건비` 등으로 분리하여 대화가 섞이지 않음
- **스레드** = 하나의 메시지에 댓글 달기. 주제별 맥락이 유지됨
- **검색** = 과거 대화도 키워드로 즉시 검색 가능
- **연동** = Google Calendar, Claude 등 외부 서비스와 자동 연결

**재무팀 현재 채널 구성:**

`#00-공지사항` · `#00-업무소통_잡담방` · `#01-결산` · `#02-출납` · `#03-등록` · `#04-인건비` · `#05-세무` · `#06-자문` · `#07-근로학생`

**이미 이렇게 쓰고 있습니다:**
- 연말정산센터를 Slack으로 운영 : 학생 근로자 7명과 민원 접수·처리·성과 기록
- Google Calendar 연동으로 재무팀 주간 일정 자동 공유
- Claude 연동으로 채널에서 직접 AI에게 업무 질의

## 2. Obsidian — 나만의 업무 노트북 🔵 기초 구축

컴퓨터에 있는 메모장을 떠올려 보세요. Obsidian은 **메모장의 업그레이드 버전**입니다. 다만 일반 메모장과 달리, 문서끼리 연결되고, AI가 읽을 수 있는 형식(마크다운)으로 저장됩니다.

**쉽게 말하면?**
- **Vault(볼트)** = 내 컴퓨터 안의 폴더 하나. 여기에 업무 관련 문서를 다 모아둠
- **마크다운(.md)** = `#` 제목, `-` 목록, `|` 표 같은 기호로 쓰는 텍스트 파일. 한글(HWP)처럼 특정 프로그램이 필요 없음
- **[[링크]]** = 문서끼리 연결. "원천세 매뉴얼"에서 클릭 한 번에 "비거주자 가이드"로 이동
- **개인 도구** = 각자 자기 Vault에서 작업. 팀 공유는 Notion으로

**왜 마크다운으로 써야 하나요?**
- HWP/PPT/PDF는 AI가 내용을 읽을 수 없음 → 마크다운은 AI가 바로 이해 가능
- 국가AI전략위원회도 2026.3.5 회의 문서를 마크다운으로 전환 발표
- **이 문서 자체가 마크다운입니다** — 별도 프로그램 없이 어디서든 열림

**Vault 폴더 구조 (업무분장 기반):**

```
📁 Vault (= 내 컴퓨터의 폴더)
├── 결산·자금 — 연결산·월결산, 자금운용, 기금운용심의
├── 등록(수입) — 등록금 수납·환불, 가상계좌, 선급장학금
├── 회계(지출) — 지출 검토·승인, 법정지출증빙, 법인카드
├── 세무 — 인건비, 연말정산, 원천세, 부가세, 세무자문
├── 프로젝트 — 퇴직연금·디지털혁신 등
└── 템플릿 — 반복 사용 문서 틀
```

**💰 비용:**

| 항목 | 월 비용 | 비고 |
|------|--------|------|
| **앱 자체** | **무료** | 모든 핵심 기능 포함 |
| **Sync (동기화)** | **$5** | PC↔모바일 등 여러 기기 간 동기화 |

## 3. Notion — 팀 공용 위키+게시판 🔴 올해 핵심

네이버 카페나 그룹웨어 게시판을 떠올려 보세요. Notion은 **게시판 + 데이터베이스 + 문서 편집기**가 합쳐진 도구입니다. 모바일에서도 PC에서도 동일하게 접근 가능합니다.

**쉽게 말하면?**
- **페이지** = 웹에서 바로 보는 문서. 모바일에서 링크 클릭만으로 열림
- **데이터베이스** = 엑셀처럼 필터·정렬이 되는 문서 목록. 매뉴얼을 카테고리별로 관리
- **팀 워크스페이스** = 재무팀 전용 공간(dgu-finance). 팀원 전원이 접근

**Obsidian과 뭐가 다른가요?**

| | Obsidian | Notion |
|---|---|---|
| 비유 | 나만의 노트북 | 팀 공용 게시판 |
| 누가 봄 | 나만 봄 | 팀원 전체·교직원 |
| 저장 위치 | 내 컴퓨터 | 클라우드 (인터넷) |
| 용도 | 초안 작성, 공부, 메모 | 공식 매뉴얼, FAQ, 공지 |

**도구 간 흐름:**

```
Obsidian (내가 초안 작성) → Notion (팀에 공유) → Slack (링크 전달) → Claude (AI가 활용)
```

**이미 이렇게 쓰고 있습니다:**
- **재무팀 지식 라이브러리** : 업무 매뉴얼 + 교직원 FAQ를 한곳에 모은 허브
- **재무팀 안내 포털** : 업무분장·연락처·절차 안내. 연말정산 FAQ를 구글폼 상단에 삽입하여 배포
- **매뉴얼 데이터베이스 (구축 중)** : 업무영역·담당자·상태별로 매뉴얼을 검색·필터링

## 4. Claude — 똑똑한 신입 직원 🟣 시범 예정

ChatGPT를 써보셨다면 비슷합니다. Claude는 특히 **긴 문서를 읽고 이해하는 데 강점**이 있어서, 업무 매뉴얼을 통째로 학습시킨 뒤 질문하면 관련 내용을 찾아 답변해줍니다.

**쉽게 말하면?**
- 매뉴얼 100페이지를 읽혀놓고 "시간강사 원천징수 어떻게 해?" 물으면 → 관련 부분을 찾아서 답변 초안을 작성
- Slack 채널에서 직접 호출 가능 → 별도 프로그램 전환 없이 업무 중에 바로 질의
- 세법 검토 도구도 AI로 개발 → 세무자문 DB를 바탕으로 각종 세법 관련 사례 검토 자동화

**핵심 원칙:**
- 교직원이 AI를 직접 쓰는 것이 아님
- **재무팀이 AI로 답변 초안 생성** → 전문가가 검토·보완 → 최종 답변 제공
- 개인정보(주민번호, 급여 등)는 절대 AI에 입력 금지
